# Hello World: My First Post

Welcome to my new blog! This is a simple, markdown-powered blog where I can share my thoughts, research findings, and technical deep-dives easily.

## Why a Markdown Blog?

As an engineer, writing in Markdown is second nature. Setting up a heavy Content Management System (CMS) like WordPress or Ghost felt like overkill for a personal portfolio. 

Instead, this blog works by simply:
1. Writing a `.md` file in my code editor.
2. Dropping it into the `posts/` folder.
3. Adding a link on the main `blog.html` page.

The `marked.js` library handles the rest dynamically on the frontend.

I'll be posting about:

- **Robotics:** Experiments with MuJoCo, Isaac Sim, and RL policies.
